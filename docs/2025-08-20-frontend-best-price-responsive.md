# Frontend: правки адаптивности блока «Лучшая цена» — 2025‑08‑20

Исправлена адаптивная верстка карусели «Лучшая цена» на главной: карточки больше не «съезжают», корректно масштабируются, элементы не перекрываются.

## Что изменено
- Файл: `protekauto-frontend/src/styles/my.css`
  - Улучшено горизонтальное пролистывание: `scroll-snap-type`, touch‑инерция, padding.
  - Стабилизированы размеры карточек: `clamp()` для ширины и `scroll-snap-align`.
  - Изображения: `object-fit: contain`, фиксированная высота контейнера по брейкпоинтам.
  - Бейдж скидки (`.saletagbp`) закреплен в верхнем левом углу изображения (absolute).
  - На мобильных скрыты стрелки карусели, оставлен скролл свайпом.

## Как проверить
1) Откройте главную страницу на ширинах 1440, 1024, 768, 480, 360.
2) Пролистайте карусель «Лучшая цена»: карточки ровно выравниваются по щелчкам.
3) Бейдж скидки не смещается, изображения не «растягиваются».

## Примечания
- Верстка использует существующие классы Webflow; правки внесены через `my.css`, который подключается после `protekproject.webflow.css`.
